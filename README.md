# 🍕 Pizza Sales Data Analysis

# 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights. The analysis helps understand customer preferences, sales performance, and product trends to support data-driven decision-making.

---

# 🎯 Objectives

* Analyze overall sales performance
* Identify top-selling and high-revenue pizzas
* Understand customer ordering patterns
* Evaluate category-wise and size-wise demand
* Track sales trends over time

---

# 📊 Key Performance Indicators (KPIs)

* **Total Orders:** Total number of orders placed
* **Total Revenue:** Revenue generated from pizza sales
* **Highest-Priced Pizza:** Premium product identification
* **Most Common Pizza Size:** Customer preference analysis
* **Top 5 Pizzas:** Best-selling items based on quantity

---

# 📈 Analysis & Insights

# 🕒 Sales Trends

* Identified peak ordering hours using time-based analysis
* Evaluated daily order volume and average pizzas sold per day

# 🍕 Product Performance

* Determined top-selling pizza types by quantity
* Analyzed highest revenue-generating pizzas
* Identified most popular pizza sizes

# 📦 Category Insights

* Compared sales across different pizza categories
* Calculated category-wise contribution to total revenue

### 📉 Advanced Analytics

* **Revenue Contribution (%):** each category in total sales
* **Cumulative Revenue:** Running total of revenue over time
* **Top 3 Pizzas per Category:** SQL window functions (RANK)

---

## 🛠️ Tools & Technologies

* **SQL (Joins, Aggregations, Subqueries, Window Functions)
* Relational Database (MySQL)

---

## 🧠 Key SQL Concepts Used

* INNER JOIN
* GROUP BY & ORDER BY
* Aggregate Functions (SUM, COUNT, AVG)
* Subqueries
* Window Functions (RANK, OVER)
